# Validation Status Artifacts

This directory stores machine-readable developer evidence for DRM.jl validation
lanes. These files are evidence ledgers, not test runners and not public
capability claims.

## Location-Only Gaussian REML Row Contract

`2026-06-21-loconly-reml-simulation-status.tsv` is the stable row artifact for
the exact-Gaussian location-only phylogenetic mean REML diagnostic lane. It is
regenerated by:

```sh
julia --project=. tools/loconly-reml-simulation-status.jl \
  --output docs/dev-log/validation-status/2026-06-21-loconly-reml-simulation-status.tsv
```

The default artifact has four rows:

- `stable_recovery`
- `condition_grid`
- `weak_signal_boundary_probe`
- `larger_interior_stress`

The schema is owned by `_loconly_reml_simulation_status_schema()` and guarded by
`test/test_location_only_reml_mme.jl`. If a column changes, update the helper,
the test's explicit schema tuple, this README, the TSV artifact, and the
paired check-log/after-task evidence in the same change.

Optional stress rows are deliberately opt-in. `--with-medium-stress` runs the
medium stress smoke. `--with-large-stress` adds a large-stress row, but reports
`large_interior_stress_skipped` unless an explicit
`--large-stress-budget-seconds` value meets the runtime guard. That keeps the
default row writer cheap while preserving a machine-readable place for larger
developer evidence.

Every row in this lane must keep:

- `target = gaussian_loconly_phylo_reml`
- `coverage_status = not_evaluated`
- `mcse_status = diagnostic_only`
- `ai_reml_ready = false`

This is exact-Gaussian developer evidence only. It is not q4 evidence, not
non-Gaussian evidence, not R bridge promotion, not interval coverage evidence,
and not an Ayumi-facing claim.

## Location-Only Gaussian REML External Comparator Probe

`2026-06-22-loconly-reml-external-comparator-probe.tsv` records the optional
developer package/version probe for the phylolm-style same-estimand comparator
candidate. It is regenerated by:

```sh
julia --project=. tools/loconly-reml-external-comparator-probe.jl \
  --output docs/dev-log/validation-status/2026-06-22-loconly-reml-external-comparator-probe.tsv
```

The artifact may record a local package version when `Rscript` and the
candidate package are installed. That is package-availability evidence only.
Every row must keep:

- `target = gaussian_loconly_phylo_reml`
- `fit_status = not_run`
- `same_estimand_status = requires_fixture_reproduction`
- `dependency_status = not_added`
- `coverage_status = not_evaluated`
- `ai_reml_ready = false`

This artifact does not run the external comparator, compare estimates, add a
dependency, promote the R bridge, evaluate interval coverage, or mark AI-REML
ready.
