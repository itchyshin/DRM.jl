## What was implemented

- **`drm(...; method::Symbol = :ML)`** on the Gaussian front end. `:ML` is the
  default and unchanged; `:REML` routes the fixed-effect location–scale cell to a
  new `_fit_fixed_gaussian_reml`. Any other cell (random effects, structured /
  phylo / meta, bivariate) raises a clear `ArgumentError` under `:REML`.
- **`_fit_fixed_gaussian_reml`** — standard Patterson–Thompson / Harville REML for
  the heteroscedastic Gaussian `y_i ~ N(Xμ_i β_μ, σ_i²)`, `log σ_i = Xσ_i β_σ`:
  profile β_μ out by weighted least squares (`W = diag(σ_i⁻²)`), optimise the
  restricted objective `ℓ_R = ℓ_P − ½ logdet(Xμ'WXμ) + (pμ/2)log2π` over β_σ.
- **`DrmFit`** gained `estim_method::Symbol`, `reml_loglik::Float64`,
  `ml_loglik::Float64`. All existing constructors are preserved via the 11-arg
  outer constructor (defaults `:ML` / `NaN` / `loglik`); a new `_withreml` helper
  marks REML fits. Accessors `estimation_method`, `reml_loglik`, `ml_loglik`
  exported; `loglik` docstring carries the cross-structure caveat.
- **Model-selection guard.** `aic`/`bic` on a REML fit emit a one-time `@warn`
  (the value is only valid for variance-only comparisons). `lrtest`/`anova`
  **error** when comparing REML fits with different fixed-effect (mean) structures
  (`_mean_structure` compares the `:mu` coefnames) and stay silent for a
  variance-only difference or any all-ML comparison.

## Math derivation (by hand, pending CI)

Homoscedastic special case (`sigma ~ 1`), the cleanest acceptance anchor:
- ML σ-stationarity: `∂/∂βσ [n βσ + ½ SSR e^{-2βσ}] = 0 ⇒ σ²_ML = SSR / n`.
- REML adds `+½ logdet(Xμ'WXμ) = +½(-2 pμ βσ + logdet Xμ'Xμ)`, so the σ-part is
  `(n - pμ) βσ + ½ SSR e^{-2βσ} + c ⇒ σ²_REML = SSR / (n - pμ) > σ²_ML`.
- Hence `σ²_REML / σ²_ML = n / (n - pμ)` exactly — the n vs n−p divisor. This is
  the location-scale analog of the q4 result `diag(Λ_REML) ≥ diag(Λ_ML)`.
- β̂_μ is unchanged at the optimum (the WLS estimate at the REML β_σ).

## Tests added (`test/test_reml.jl`)

1. **ML unchanged** — `method = :ML` byte-for-byte vs no kwarg (coef + loglik
   equal); default fit reports `:ML`, `reml_loglik` NaN, `ml_loglik == loglik`.
2. **REML ≥ ML property** — `σ²_REML > σ²_ML`, ratio `≈ n/(n-pμ)` (rtol 1e-4),
   β̂_μ unchanged (rtol 1e-5), `ml_loglik(reml) ≤ loglik(ml)`. Plus a
   heteroscedastic check (`sigma ~ 1 + x`): REML raises the log-σ intercept.
3. **FD-gradient self-consistency** — an independent rebuild of the restricted
   objective has `max|∇| < 1e-4` at the fitted β̂_σ.
4. **Guard** — cross-mean-structure REML `lrtest`/`anova` throws; variance-only
   REML and all-ML cross-mean comparisons return a NamedTuple; `aic`/`bic` on a
   REML fit are finite.
5. **Rejections** — RE-on-mean under `:REML` and an unknown `method` symbol both
   throw `ArgumentError`.
