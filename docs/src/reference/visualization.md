# Visualization

!!! note "Status — Reference"
    Mirrors drmTMB's [Visualization](https://itchyshin.github.io/drmTMB/reference/index.html) (3 in drmTMB). **In DRM.jl today:** the backend-free plotting *data providers* — `parameter_surface` (2-D profile-deviance grid, the data behind `plot_parameter_surface`) and `corpairs_data` (between-response residual-correlation summary, behind `plot_corpairs`). The base package carries no plotting dependency; render the returned grids/vectors with any backend (Makie / Plots) in a few lines.
